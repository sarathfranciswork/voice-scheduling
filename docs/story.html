The Vision

What if scheduling a vaccine
was as easy as a phone call?

CVS Voice Scheduling reimagines the appointment experience. No forms, no dropdowns, no login walls — just a natural conversation with an AI that handles everything from eligibility to confirmation.

🎙️

Voice-First Design

Talk naturally to schedule. The AI understands context, asks clarifying questions, and walks through the 10-step CVS scheduling flow conversationally.

🏥

Real CVS Integration

Not a mockup. Live integration with CVS.com APIs for real-time vaccine eligibility, store availability, time slots, and confirmed appointments.

📱

Native iOS Experience

SwiftUI app with dynamic surfaces — interactive cards for store selection, time grids, patient forms, and questionnaires render natively in-chat.

🧠

Agentic Architecture

MCP protocol bridges LLM intelligence to real-world APIs. The agent autonomously chains tools, caches session state, and handles error recovery.

Multi-Modal Experience

Three ways to interact

Switch seamlessly between chat, voice, and video mid-conversation — context preserved, no re-greeting.

💬 Text Chat

Real-time streaming chat powered by ElevenLabs Agent Platform over WebSocket. Dynamic UI widgets render inline for rich interactions.

  • ✓ ElevenLabs text-only WebSocket protocol
  • ✓ Dynamic surfaces (store cards, time grids, forms)
  • ✓ Vision API for document scanning
  • ✓ Hold-to-talk speech input

📞 Voice Call

Full-duplex voice conversation via ElevenLabs Conversation SDK over LiveKit WebRTC. Natural speech with real-time transcription.

  • ✓ ElevenLabs ConvAI SDK (LiveKit WebRTC)
  • ✓ Real-time speech transcription
  • ✓ Mute control & agent state tracking
  • ✓ Context preservation from chat

🎥 Video Avatar

Face-to-face conversation with a HeyGen LiveAvatar. ElevenLabs agent provides the voice while HeyGen renders the visual avatar in real-time.

  • ✓ HeyGen LiveAvatar LITE (LiveKit)
  • ✓ ElevenLabs Agent ↔ HeyGen server-to-server
  • ✓ Apple SFSpeechRecognizer transcription
  • ✓ Floating bubble ↔ full-screen toggle
Architecture

Five-layer agentic stack

From native iOS UI to live CVS API calls — each layer is purpose-built, async-first, and fully autonomous.

Layer 1 — Client
iOS App + React Frontend
SwiftUI + ElevenLabs SDK + HeyGen LiveKit  |  React 19 + Vite + WebSocket
Layer 2 — API Gateway
FastAPI Backend
29 endpoints  |  WebSocket streaming  |  aiosqlite  |  Dynamic Surfaces
Layer 3 — Agent Bridge
MCP Server (FastMCP)
25 tools  |  HTTP transport on port 8001  |  Session-scoped caching
Layer 4 — API Client
CVS API Wrapper (httpx + Playwright)
Async client  |  Guest token lifecycle  |  Akamai stealth fingerprinting
Layer 5 — External
CVS.com Production APIs
Live eligibility  |  Store search  |  Real-time availability  |  Confirmed bookings

MCP Tool Catalog

25 tools bridging AI to pharmacy

Each tool is a structured function the AI agent can call to interact with CVS systems — from checking vaccine eligibility to confirming appointments.

💉 Vaccines 3 tools

Eligibility checking by date of birth and vaccine type

  • get_eligible_vaccines(dob)
  • check_vaccine_eligibility(dob, codes)
  • refresh_session()

🏪 Stores 3 tools

Location search, availability, and time slot discovery

  • search_stores(address, radius)
  • get_available_time_slots(date)
  • get_store_details(store_id)

📅 Scheduling 7 tools

Full appointment lifecycle from reservation to confirmation

  • soft_reserve_slot(clinic, date, time)
  • submit_patient_details(...)
  • get_questionnaire()
  • submit_questionnaire(answers)
  • get_user_schedule()
  • confirm_appointment()
  • address_typeahead(text)

🔐 Authentication 6 tools

OTP and password login flows with DOB verification

  • login_to_cvs(email, password?)
  • verify_otp(code)
  • verify_dob(month, day, year)
  • start_manual_login()
  • check_login_status()
  • logout()

👤 Profile 3 tools

Authenticated user profile and appointment management

  • get_patient_profile()
  • get_my_appointments()
  • cancel_appointment(id)

🔧 System 3 tools

Vision, session management, and surface generation

  • vision/analyze (GPT-4o)
  • dynamic_surface_builder
  • webhook_processor

End-to-End Flow

10-step scheduling journey

From "I need a flu shot" to a confirmed appointment — fully automated with per-step caching and error recovery.

1

Collect Date of Birth

Agent asks for DOB to determine vaccine eligibility by age group

2

Check Eligibility

Queries CVS eligibility API, returns available vaccines with NDC codes

get_eligible_vaccines()
3

Select Vaccine

User picks from eligible options via VaccineSelectionCard surface

check_vaccine_eligibility()
4

Find Nearby Stores

Searches CVS locations by ZIP code, returns stores with availability

search_stores()
5

Choose Time Slot

Fetches available slots (morning/afternoon/evening) for selected store

get_available_time_slots()
6

Reserve Slot

Soft-holds the appointment while collecting patient details

soft_reserve_slot()
7

Patient Details

Collects name, email, phone, address via PatientFormCard surface

submit_patient_details()
8

Health Questionnaire

Dynamic screening questions rendered as yes/no cards

get_questionnaire() → submit_questionnaire()
9

Duplicate Check

Verifies no overlapping appointments exist

get_user_schedule()
10

Confirm Booking

Finalizes appointment, shows ConfirmationSummaryCard with haptic celebration

confirm_appointment()
